claude-tmux

Claude Code plugin for tmux awareness, parallel agents, and self-continuation. View on GitHub

Install

/plugin

Then search for claude-tmux in the Discover tab.

Dev Server + Live Logs

Start a dev server in background, show logs in a visible pane, wait for ready.

Parallel Bug Fixes

Spawn multiple Claude agents to fix different bugs simultaneously.

Tests + Self-Continuation

Run tests in background, queue a reminder for the next session.

Record a Demo

Use /record to capture Claude interactions as asciinema files.

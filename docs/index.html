claude-tmux

Claude Code plugin for tmux awareness, parallel agents, and self-continuation. View on GitHub

Install

Option 1: Plugin Manager

/plugin

Then search for claude-tmux in the Discover tab.

Option 2: Add as Marketplace

/plugin marketplace add https://github.com/buremba/claude-tmux
/plugin install claude-tmux

Option 3: Manual

git clone https://github.com/buremba/claude-tmux ~/.claude/plugins/claude-tmux
